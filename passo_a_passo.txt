LOGIN COM O CADASTRO DE FUNCIONARIO:
    ADMIN - ADMIN
        ABRE TELA DE CONFIGURACAO
            CONFIGURACAO COM O DDNS OU IP DA MAQUINA COM O SERVER RODANDO
    BUSCA OS DADOS DE PRODUTOS NO SERVIDOR
    LOGIN ABRE TELA DE NOVA COMANDA OU ESCOLHER ABERTA


TELA COM NOVA COMANDA E BUSCAR EXISTENTE:
    NOVA COMANDA:
        ABRE O PEDIDO VAZIO
    BUSCAR COMANDA:
        DIALOG COM COMANDAS EM ABERTO E PASSAR PARA O PEDIDO


PEDIDO:
    BUSCA DE PRODUTOS VAZIA:
        MOSTRA OS GRUPOS
    COM ALGO DIGITADO BUSCA:
        MOSTRA OS ITENS QUE CONTEM
    ARRASTAR PARA O LADO DELETA O ITEM

    SALVAR ENVIA O JSON PARA O SERVER
        SE FOR COMANDA NOVA:
            FUNCAO DE INSERIR()
        SE NAO:
            FUNCAO DE DELETAR()


FUNCOES:
    FUNCAO DE INSERIR():
        RECEBER JSON
        PEGAR CODIGO NA C000000
        SOMAR +1 PARA PROXIMOS PEDIDOS
        COMMIT
        COM O JSON CRIAR LISTA COM O CABECALHO DA COMANDA
        COM O JSON CRIAR LISTA COM OS ITENS DA COMANDA
        CRIAR INSERTS PARA CABECALHO E ITENS
        INSERIR CABECALHO
        COMMIT
        INSERIR ITENS
        COMMIT
    FUNCAO DE DELETAR():
        RECEBER JSON
        ATUALIZAR CABECALHO
        VERIFICAR TAG DELETADOS
        DELETAR ITENS NA TAG DELETADOS
